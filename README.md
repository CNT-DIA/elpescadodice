## Cómo añadir imágenes

1. Crea una carpeta con la estructura: `images/YYYY/MM/`
2. Añade tus imágenes en formato JPG, PNG, o GIF
3. Edita este README.md para añadir nuevas imágenes en orden cronológico
4. Sigue el formato: `![Descripción](images/YYYY/MM/nombre-archivo.extension)`

**Estructura de carpetas:**
```
images/
├── 2026/
│   ├── 06/
│   │   ├── imagen-01.jpg
│   │   └── imagen-02.jpg
│   └── 07/
│       └── imagen-03.jpg
```
